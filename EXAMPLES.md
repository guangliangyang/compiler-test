# 编译器演示测试用例

## 基础算术表达式

### 1. 简单加法
```
输入: 5 + 3
预期结果: 8
```

### 2. 运算符优先级
```
输入: 5 + 3 * 2
预期结果: 11 (不是 16)
```

### 3. 括号改变优先级
```
输入: (5 + 3) * 2
预期结果: 16
```

### 4. 复杂表达式
```
输入: 2 + 3 * 4 - 1
预期结果: 13
```

### 5. 幂运算
```
输入: 2^3
预期结果: 8
```

### 6. 嵌套括号
```
输入: ((2 + 3) * 4) / 2
预期结果: 10
```

### 7. 负数
```
输入: -5 + 3
预期结果: -2
```

### 8. 浮点数
```
输入: 3.14 * 2
预期结果: 6.28
```

## 错误处理测试

### 1. 括号不匹配
```
输入: (5 + 3
预期: 语法错误
```

### 2. 缺少操作数
```
输入: 5 +
预期: 语法错误
```

### 3. 非法字符
```
输入: 5 & 3
预期: 词法错误
```

## AST结构验证

### 表达式: 5 + 3 * 2
```
预期AST结构:
    +
   / \
  5   *
     / \
    3   2
```

### 表达式: (5 + 3) * 2
```
预期AST结构:
    *
   / \
  +   2
 / \
5   3
```
